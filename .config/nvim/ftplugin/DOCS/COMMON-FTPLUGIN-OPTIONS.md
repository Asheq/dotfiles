Some buffer-local options that are commonly set in filetype plugins (ftplugins) include:

# Commenting

- `commentstring`
- `comments`

# Formatting

- `formatoptions`
- `formatlistpat`
- `formatprg`
- `textwidth`

# Whitespace

- `indentkeys`

# Folding

- `foldmethod`
- `foldexpr`
- `foldcolumn`

# Search

- `path`
- `suffixesadd`
- `suffixes`
- `isfname`

- `iskeyword`
- `keywordprg`

- `include`
- `includeexpr`
- `define`

# Other

- `matchpairs`
- `omnifunc`
