## Modifiers

- `<S-...>` = "Shift"
- `<C-...>` = "Control"
- `<M-...>` = "Option" on Mac, "Alt" on PC/Linux
- `<D-...>` = "Command" on Mac, "Windows" on PC, "Super" on Linux
